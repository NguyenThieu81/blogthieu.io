Khám Phá Web APIs Trong JavaScript

Nguyễn Văn Thiều |   28 December 2024

Web APIs (Application Programming Interfaces) là các giao diện cho phép JavaScript tương tác với các tính năng của trình duyệt và các dịch vụ bên ngoài. Chúng cung cấp các phương thức và thuộc tính để thực hiện các tác vụ như thao tác với DOM, xử lý sự kiện, và giao tiếp với máy chủ. Trong hướng dẫn này, bạn sẽ tìm hiểu về một số Web APIs phổ biến và cách sử dụng chúng trong JavaScript

Lưu ý:Để xem thêm các hướng dẫn khác, hãy truy cập Hướng dẫn JavaScript trên W3Schools.

Yêu cầu

  • Trình duyệt web Bất kỳ trình duyệt hiện đại nào (Chrome, Firefox, Safari, Edge) đều có thể chạy JavaScript..
  • Công cụ để chỉnh sửa mã. Bất kỳ trình soạn thảo văn bản hoặc IDE (như IntelliJ IDEA, Eclipse hoặc NetBeans) nào cũng đều phù hợp..

Các Web APIs Phổ Biến

  1. DOM API DOM API cho phép bạn tương tác với cấu trúc của tài liệu HTML. Bạn có thể thêm, xóa, và thay đổi các phần tử trong DOM.

Ví dụ:

// Thêm một phần tử mới vào DOM
const newElement = document.createElement("p");
newElement.innerText = "This is a new paragraph.";
document.body.appendChild(newElement);
  1. Fetch API Fetch API cho phép bạn thực hiện các yêu cầu HTTP để lấy dữ liệu từ máy chủ. Đây là cách hiện đại để thực hiện AJAX.

Ví dụ:

fetch("https://jsonplaceholder.typicode.com/posts")
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error("Error:", error));
  1. Geolocation API Geolocation API cho phép bạn truy cập vị trí địa lý của người dùng. Điều này rất hữu ích cho các ứng dụng cần thông tin vị trí.

Ví dụ:

if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(position => {
        console.log("Latitude:", position.coords.latitude);
        console.log("Longitude:", position.coords.longitude);
    }, error => {
        console.error("Error getting location:", error);
    });
} else {
    console.log("Geolocation is not supported by this browser.");
}
  1. Web Storage API Web Storage API cho phép bạn lưu trữ dữ liệu trên trình duyệt của người dùng. Có hai loại lưu trữ: localStorage và sessionStorage.

Ví dụ:

// Lưu trữ dữ liệu
localStorage.setItem("username", "JohnDoe");

// Lấy dữ liệu
const username = localStorage.getItem("username");
console.log(username); // Đầu ra: JohnDoe

// Xóa dữ liệu
localStorage.removeItem("username");
  1. Canvas API Canvas API cho phép bạn vẽ đồ họa 2D và 3D trên trang web. Bạn có thể tạo ra các hình ảnh động, trò chơi, và nhiều ứng dụng đồ họa khác.

Ví dụ:

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
    const canvas = document.getElementById("myCanvas");
    const ctx = canvas.getContext("2d");
    ctx.fillStyle = "blue";
    ctx.fillRect(20, 20, 150, 50); // Vẽ hình chữ nhật
</script>
  1. WebSockets API WebSockets API cho phép bạn tạo kết nối hai chiều giữa máy khách và máy chủ, giúp truyền tải dữ liệu theo thời gian thực.

Ví dụ:

const socket = new WebSocket("ws://example.com/socket");

socket.onopen = function() {
    console.log("Connection established.");
    socket.send("Hello Server!");
};

socket.onmessage = function(event) {
    console.log("Message from server:", event.data);
};
