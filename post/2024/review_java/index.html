Yêu cầu

  • Cài đặt Java Development Kit (JDK). Đảm bảo bạn đã cài đặt JDK 8 hoặc phiên bản mới hơn.
  • Công cụ để chỉnh sửa mã. Bất kỳ trình soạn thảo văn bản hoặc IDE (như IntelliJ IDEA, Eclipse hoặc NetBeans) nào cũng đều phù hợp..
  • Một terminal. Java hoạt động tốt trên bất kỳ terminal nào trên Linux và Mac, cũng như trên Command Prompt hoặc PowerShell trong Windows

Hãy chắc chắn rằng bạn đã cài đặt JDK từ trang chính thức của Oracle..

Thiết Lập Môi Trường Phát Triển Java

Để bắt đầu, hãy thiết lập môi trường phát triển Java của bạn.

  1. Tải và Cài Đặt JDK

    Truy cập trang tải JDK của Oracle..

    Làm theo hướng dẫn cài đặt cho hệ điều hành của bạn.

  2. Thiết Lập Biến Môi Trường

    Trên Windows: Nhấp chuột phải vào ‘This PC’ hoặc ‘My Computer’ và chọn ‘Properties’. Nhấp vào ‘Advanced system settings’ và sau đó ‘Environment Variables’. Dưới ‘System variables’, tìm biến Path và thêm đường dẫn đến thư mục bin của JDK (ví dụ: C:\Program Files\Java\jdk-11\bin).

Trên Linux hoặc Mac: Mở terminal và thêm dòng sau vào .bashrc hoặc .bash_profile:

    export PATH=$PATH:/path/to/jdk/bin

  1. Xác Minh Cài Đặt Mở Command Prompt hoặc terminal và chạy:

    java -version

Bạn sẽ thấy phiên bản Java đã cài đặt.

$ go get golang.org/x/example

Viết Chương Trình Java Đầu Tiên

Bây giờ, hãy viết một chương trình Java đơn giản.

  1. Tạo Thư Mục Cho Mã Của Bạn.

Mở Command Prompt hoặc terminal và tạo một thư mục cho các dự án Java của bạn:

mkdir java_workspace
cd java_workspace
  1. Tạo Tệp Java.

Tạo một tệp mới có tên HelloWorld.java bằng trình soạn thảo văn bản của bạn:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  1. Biên Dịch Chương Trình Java.

Trong terminal, điều hướng đến thư mục nơi bạn đã lưu HelloWorld.java và chạy:

javac HelloWorld.java

Lệnh này biên dịch tệp Java và tạo ra tệp HelloWorld.class. 4. Chạy Chương Trình Java.

Thực thi chương trình bằng cách chạy:

java HelloWorld

Bạn sẽ thấy đầu ra:

Hello, World!

. int cho số nguyên . double cho số thực . char cho ký tự . boolean cho giá trị đúng/sai
